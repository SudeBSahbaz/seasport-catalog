# SeaSport Catalog Uygulaması

SeaSport Catalog, Flutter kullanılarak geliştirilmiş modern bir mobil katalog uygulamasıdır.  
Uygulama kürek (rowing) ve stand-up paddle (SUP) ekipmanlarını listelemek amacıyla hazırlanmıştır.

---

# Proje Açıklaması

Bu uygulama kullanıcıların su sporları ürünlerini görüntüleyebilmesi için geliştirilmiştir.  
Ürünler kart yapısında listelenmekte ve kullanıcılar ürün detaylarını inceleyebilmektedir.

Uygulama içerisinde:

- Ürün listeleme
- Ürün arama
- Ürün detay sayfası
- Sepete ekleme simülasyonu
- JSON veri kullanımı
- Modern kullanıcı arayüzü

özellikleri bulunmaktadır.

---

# Özellikler

- Responsive ürün grid yapısı
- Ürün detay sayfası
- Search (arama) özelliği
- Cart counter (sepet sayacı)
- JSON tabanlı veri kullanımı
- Material Design arayüzü
- Flutter navigation sistemi

---

# Kullanılan Teknolojiler

- Flutter
- Dart
- Material Design
- JSON
